Et je code le samedi soir parce que c'est la que je suis le plus moi.

### Comment je fonctionne

Chaque projet suit le meme cadre : **DDD + Clean Architecture**. Interfaces → application → domain → infrastructure. TDD obligatoire. Factories partagees. Conventional Commits. Pas de catch silencieux, pas de `any`, pas de logique metier dans les controllers.

Mon CLAUDE.md fait 200 lignes. C'est le contrat entre moi et l'IA. Le cadre qui permet a un module DDD complet de sortir en 18 minutes — avec les tests.

La rigueur ne ralentit pas. La rigueur est ce qui permet d'accelerer.
